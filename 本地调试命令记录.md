# 本地调试命令记录

## 运行环境
node: `22`

## 获取原码

- 项目地址
  ```bash
  https://github.com/redTreeOnWall/memorains
  ```
- 下载编译好的代码
  ```bash
  curl -L -o 代码.tar.gz https://github.com/redTreeOnWall/memorains/releases/download/0.8.66/package.tar.gz
  ```
- 解压代码
  ```bash
  tar -xzvf 代码.tar.gz
  ```
- 重命名
  ```bash
  mv package 代码
  ```

## 调试运行

> 采用 compose 方式部署调试容器

- 容器
  - 文件 [compose-dev.yaml](compose-dev.yaml)
  - 运行 compose
    ```bash
    docker compose -f compose-dev.yaml up -d
    ```
- [修改代码](汉化和修改.md)
- 进入容器 memorains 终端
  ```bash
  docker exec -it memorains /bin/ash
  ```
- 基础配置
  - 设置 alpine 源
    ```bash
    sed -i 's/dl-cdn.alpinelinux.org/mirrors.tuna.tsinghua.edu.cn/g' /etc/apk/repositories && \
    sed -i 's/https:/http:/g' /etc/apk/repositories
    ```
  - 设置 npm 源
    ```bash
    npm config set registry https://registry.npmmirror.com
    ```
- 运行服务端
  - 进入目录
    ```bash
    cd server
    ```
  - 安装依赖
    ```bash
    npm i
    ```
  - 运行
    ```bash
    node build/index.js
    ```
- 运行前段
  - 自签证书
    - 创建目录
      ```bash
      mkdir -p 代码/certificate
      ```
    - 生成证书
      ```bash
      openssl req -x509 -newkey rsa:4096 \
        -keyout 代码/certificate/cert.key \
        -out 代码/certificate/cert.pem \
        -days 3660 \
        -nodes \
        -subj "/C=CN/ST=State/L=City/O=Organization/CN=localhost"
      ```
  - 打开新终端，容器 memorains 终端
    ```bash
    docker exec -it memorains /bin/ash
    ```
  - 安装 nginx
    ```bash
    apk update && apk add --no-cache nginx
    ```
  - 修改 nginx 配置文件
  - 启动 nginx
    ```bash
    nginx -c /app/nginx.conf -g 'daemon off;'
    ```
- 删除容器
  ```bash
  docker compose -f compose-dev.yaml down
  ```
- 编写容器启动脚本 [docker-entrypoint.sh](docker-entrypoint.sh)
